## 📝 Troubleshooting

If the script fails to run:

1. Right-click the Start button and select "Windows PowerShell (Admin)".
2. Type `Set-ExecutionPolicy RemoteSigned` and press Enter.
3. Type `Y` and press Enter.
4. Try running the script again.

If you suspect a setting caused a conflict:

1. Re-run the script. Many versions of the tool include an option to "Reset to Default" or "Restore Settings". 
2. Ensure no other optimization software is running at the same time. Having two separate programs changing the same settings can lead to conflicts.
